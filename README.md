# Encriptador de Texto

Este proyecto es un **encriptador y desencriptador de texto desarrollado con HTML, CSS y JavaScript**, creado como parte del proceso de aprendizaje en desarrollo web frontend.

Permite transformar mensajes usando reglas de sustitución de caracteres y recuperar el texto original de forma sencilla mediante una interfaz web amigable.

---

## Objetivo del proyecto

El objetivo principal de este proyecto es **practicar fundamentos de JavaScript y manipulación del DOM**, así como mejorar habilidades en:

- Lógica de programación.
- Eventos en JavaScript.
- Interacción con el usuario.
- Diseño básico de interfaces web.
- Buenas prácticas de estructura HTML y estilos CSS.

---

## Funcionalidades principales

- Encriptar texto ingresado por el usuario.
- Desencriptar mensajes previamente encriptados.
- Copiar el texto resultante al portapapeles.
- Validación básica de entrada de texto.
- Interfaz web simple y responsiva.

---

## Tecnologías utilizadas

- **CSS3**
- **JavaScript (DOM y eventos)**
